# piscine-c

> Exercices de la Piscine C — École 42

## Description

Ensemble d'exercices quotidiens de la Piscine C (jours j00 à j13) : premiers pas en C, boucles, conditions, pointeurs, fonctions, structures et petits programmes.

Chaque dossier `jXX/` contient les exercices du jour (`ex00`, `ex01`, …) avec leurs sujets respectifs.

## Structure

```
j00/ … j13/    — exercices par jour
j10/           — exercices avancés (includes, Makefile)
j12/           — exercices avec Makefile
```

## Compilation

La plupart des exercices se compilent individuellement :

```bash
gcc -Wall -Wextra -Werror fichier.c
```

Certains dossiers (j10, j12) disposent d'un `Makefile`.

## Compétences

`C` · `Algorithmique` · `Bases de la programmation`
